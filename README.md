# 🎬 gifly-lite

A lightweight **Giphy-inspired GIF browsing app** built with React and Vite.
Explore trending GIFs, search by keywords, browse categories, and save your favorites — all in a fast, minimal interface.

---

## 🚀 Features

* 🔍 Search GIFs by keyword
* 📂 Browse GIFs by category
* 🎞️ View individual GIF pages
* ❤️ Save and manage favorites
* ⚡ Fast performance with Vite
* 🧭 Smooth client-side routing

---

## 🛠️ Tech Stack

* React
* Vite
* React Router
* Tailwind CSS

---

---

## 🧭 Routes

| Route            | Description             |
| ---------------- | ----------------------- |
| `/`              | Home page               |
| `/:category`     | Browse GIFs by category |
| `/search/:query` | Search results          |
| `/:type/:slug`   | Single GIF view         |
| `/favorites`     | Favorite GIFs           |

---

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/gifly-lite.git
cd gifly-lite
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run dev
```
